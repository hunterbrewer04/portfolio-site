## A Quick Code Taste

Here is the kind of thing you will find here. A small TypeScript utility I reached for last week — a type-safe `groupBy` that actually preserves the value type:

```ts
function groupBy<T, K extends string>(
  items: T[],
  keyFn: (item: T) => K
): Record<K, T[]> {
  return items.reduce(
    (acc, item) => {
      const key = keyFn(item);
      (acc[key] ??= []).push(item);
      return acc;
    },
    {} as Record<K, T[]>
  );
}

// Usage
const posts = [
  { slug: "hello-world", tag: "intro" },
  { slug: "typescript-tips", tag: "typescript" },
  { slug: "getting-started", tag: "intro" },
];

const byTag = groupBy(posts, (p) => p.tag);
// { intro: [...], typescript: [...] }
```

Nothing earth-shattering, but the kind of thing worth writing down so future-me does not spend another twenty minutes reinventing it.
